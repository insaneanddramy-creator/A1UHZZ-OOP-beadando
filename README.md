# A1UHZZ-OOP-beadando
Juhász Dóra - A1UHZZ, Objektumorientált programozás beadandó

Autókölcsönző Rendszer

Egyszerű konzolos autókölcsönző rendszer Python nyelven, objektumorientált programozási elvek alkalmazásával.

Funkciók:
- Autók listázása
- Autó bérlése
- Bérlés lemondása
- Aktív bérlések listázása

A program futtatásához a main.py fájl indításával lehetséges.

Fájlok:
auto.py
szemelyauto.py
teherauto.py
berles.py
autokolcsonzo.py
main.py
